MrTurno API

Bienvenido a la documentación para integradores de MrTurno. Aquí encontrarás una guía clara para autenticarte, consultar profesionales y turnos, y reservar/cancelar turnos de forma segura e idempotente.

TL;DR: Si solo querés empezar, andá a Guía rápida.

Alcance

  • Autenticación basada en JWT (Bearer).
  • Endpoints para instituciones, profesionales, prácticas y turnos.
  • Convenciones de paginación, filtros, errores y límites.

Entornos

Definí variables de entorno para que los ejemplos funcionen tal cual.

ALPHA_BASE_URL=https://api.alpha.mrturno.com/v1
PROD_BASE_URL=https://api.mrturno.com/v1
BASE_URL=$ALPHA_BASE_URL  # Cambiá a PROD cuando corresponda
JWT="<tu-token-jwt>"

Versionado

La API está versionada por path: /v1. Los cambios compatibles se comunican en el Changelog y las deprecaciones tienen un período mínimo de 90 días.

Cómo leer esta guía

  • Guía rápida: flujo de punta a punta con cURL.
  • Autenticación: cómo obtener y usar el JWT.
  • Convenciones y mejores prácticas: paginación, filtros, idempotencia, cabeceras comunes.
  • Referencia: renderizada desde openapi.yaml con Redoc o Swagger UI.
