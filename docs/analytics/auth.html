Guía de Autenticación

Utiliza la autenticación OAuth 2.0 con el flujo Client Credentials para solicitar tokens.

Solicitud de token

POST /oauth/token
Host: analytics.rentasoft.com
Content-Type: application/json

{
  "client_id": "tu-client-id",
  "client_secret": "tu-client-secret",
  "audience": "analytics-api",
  "grant_type": "client_credentials"
}

El token tendrá una vigencia de 1 hora. Renueva el token antes de su expiración.
